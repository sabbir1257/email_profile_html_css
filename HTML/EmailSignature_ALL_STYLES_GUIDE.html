üì± Installation Instructions

For Gmail:

  1. Click ‚öôÔ∏è Settings ‚Üí "See all settings"

  5. Copy the signature (Ctrl+C or Cmd+C)

  2. Click "New"
  3. Open HTML file in browser and copy signature
  4. Paste into Outlook editor
  5. Click "OK"

For Outlook Web:

  1. Settings ‚öôÔ∏è ‚Üí "View all Outlook settings"

  4. Paste into signature box
  5. Click "Save"

For Apple Mail:

  1. Mail ‚Üí Settings ‚Üí Signatures
  2. Create new signature
  3. Copy signature from HTML file
  4. Paste into signature area
  5. Uncheck "Always match my default message font"
